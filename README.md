🦷 BrightSmile Dental Clinic

A modern, conversion-focused dental clinic website built with React and Tailwind CSS.

This project demonstrates clean healthcare UI design, responsive layout structure, and user-focused appointment flow.

🛠 Tech Stack

React (Vite)

Tailwind CSS

Component-Based Architecture

Responsive Design

Google Maps Integration

✨ Features

Clean and professional medical UI

Fully responsive layout (mobile, tablet, desktop)

Appointment booking form UI

Smooth scroll navigation

Google Maps embedded contact section

Conversion-focused CTA placement

📂 Project Structure
src/
 ├── components/
 ├── hooks/
 ├── assets/
 ├── App.jsx
 ├── main.jsx
📌 Purpose

This project was built as a frontend showcase for a local healthcare business website, focusing on:

Trust-building design

Clear call-to-action hierarchy

Clean typography system

Structured section layout
